# UIScreen

## 概要
UIScreenはハードウェアベースのディスプレイに関連するプロパティを定義します。

## 関連クラス
NSObject
 
## 主要プロパティ

|プロパティ名|説明|サンプル|
|---|---|---|
|main | デバイスの画面を表すスクリーンオブジェクトを返す <br>（クラスプロパティ） | `UIScreen.main` |
|bounds | 画面の矩形（`CGRect`） | `UIScreen.main.bounds` |

## フレームワーク
UIKit.framework

## サポートOSバージョン
iOS2.0以上
